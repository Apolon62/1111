# iCloud AutoLocker — Telegram бот для автоматической блокировки iCloud устройств

Бот позволяет воркерам получать iCloud аккаунты, автоматически отслеживать входы в Настройки iCloud, менять пароль и передавать данные админу для удалённой блокировки устройств.

## Возможности

### Для воркеров
- Получение аккаунта (логин + пароль) с автоматическим запуском мониторинга
- Продление таймера сессии (до 4 часов)
- Просмотр статистики и истории локов
- Уведомления о типе входа: App Store (неудачно) или Настройки (успех, пароль сменён, ID лога)

### Для админов
- База улова — список всех заблокированных устройств
- Архив разблокированных устройств
- Поиск устройства по ID лога
- Управление устройствами: стереть, установить код-пароль, удалить с аккаунта, звук, обновить локацию
- Управление воркерами (добавление, блокировка, статистика, рассылка)
- Импорт аккаунтов (файл / вручную)
- Настройка контакта админа, таймеров, интервалов мониторинга

### Автоматика (ядро автолокера)
- Мониторинг аккаунта каждые 30–120 секунд через `icloudpy`
- Детекция типа входа (App Store vs Настройки) по геолокации и статусу устройства
- При входе в Настройки — автоматическая смена пароля через Playwright на `account.apple.com`
- Сохранение данных устройства (модель, серийный номер, IMEI, геолокация) в базу
- Генерация ID лога и уведомление воркера и админов

## Технологии

- Python 3.11
- aiogram 3.x — Telegram Bot API
- icloudpy — быстрая проверка устройств
- Playwright — веб-автоматизация (смена пароля, блокировка)
- Supabase (PostgreSQL + Realtime) — база данных и уведомления
- Railway + Docker — хостинг
